# 🍎 FruiSnap – AI Fruit Predictor

FruiSnap is a modern AI-powered web application that detects fruits from images and displays results with an interactive 3D visualization.

---

## 🚀 Features

- 🧠 AI-based fruit classification (Apple / Orange)
- 🌐 Live deployed web app (Render)
- 🖼️ Image upload + preview
- 🧊 3D fruit model visualization (Three.js)
- 🌌 Animated space background
- 📱 Mobile responsive UI

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript, Three.js
- **Backend**: Flask (Python)
- **ML Model**: TensorFlow / Keras
- **Deployment**: Render

---

## 🧠 How it Works

1. User uploads an image
2. Image is preprocessed
3. TensorFlow model predicts fruit
4. Result + confidence displayed
5. 3D model rendered dynamically
